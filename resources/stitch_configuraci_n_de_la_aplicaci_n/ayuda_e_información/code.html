Regla de División 50/50

Al realizar un viaje directo entre dos clientes, el sistema aplica la política corporativa: el kilometraje se divide automáticamente en partes iguales para ambos registros de visita.

El Asistente Inteligente

01

Botón de Acción Central

El botón circular en la pantalla principal cambia según tu estado. Presiónalo para iniciar un viaje o una visita.

02

Preguntas Obligatorias

Al llegar a destino, el asistente te consultará sobre gastos de parking o peajes. No olvides completarlos para el reembolso.

03

Cierre de Visita

Antes de marcharte, confirma la finalización para que el sistema detenga el contador de tiempo y habilite el próximo viaje.
